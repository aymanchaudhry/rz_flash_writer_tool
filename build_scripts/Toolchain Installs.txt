Toolchain Installs

The follow toolchains can be used for building.

Please choose one.

Copy/Paste the commands to install a toolchain.

-----------------------------------------------------
GNU Toolchain by ARM
Version 10.2-2021.07
https://developer.arm.com/downloads/-/gnu-a
-----------------------------------------------------
cd /tmp
wget https://developer.arm.com/-/media/Files/downloads/gnu-a/10.3-2021.07/binrel/gcc-arm-10.3-2021.07-x86_64-aarch64-none-elf.tar.xz
sudo mkdir -p /opt/arm
cd /opt/arm
sudo tar xvf /tmp/gcc-arm-10.3-2021.07-x86_64-aarch64-none-elf.tar.xz
